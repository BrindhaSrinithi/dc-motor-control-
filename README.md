# dc-motor-control-

##PROJECT DESCRIPTION:
This project demonstrates DC motor control using push buttons with an Arduino Uno. Push buttons are used to start, stop, and control the direction of a DC motor through a motor driver module (L293D). The Arduino reads the button inputs and sends control signals to the motor driver, enabling safe and efficient motor operation. This project introduces digital input handling, motor driver interfacing, and basic embedded programming concepts.
##FEATURES:
1.Start and stop the DC motor using push buttons
2.Control motor direction (Forward/Reverse)
3.Simple and beginner-friendly Arduino project
4.Demonstrates motor driver interfacing
5.Easy to modify for other motor control applications
##COMPONENTS REQUIRED:
1.Arduino Uno
2.DC Motor
3.L293D Motor Driver Module
4.Push Button
5.Breadboard
6.Jumper Wires
7.USB Cable
